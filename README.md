# Captcha Generator

A simple and interactive CAPTCHA generator built with HTML, CSS, and JavaScript. This project provides a basic implementation of a CAPTCHA system that can be integrated into web applications for basic bot protection.

## Features

- Generates random alphanumeric CAPTCHA codes
- Case-sensitive verification
- Simple and clean user interface
- Responsive design
- Easy to integrate into existing projects
- Generate new CAPTCHA with a single click
- Visual feedback for verification status

## How It Works

The CAPTCHA generator creates a 6-character code consisting of:
- 3 random letters (uppercase or lowercase)
- 3 random numbers

The verification is case-sensitive, ensuring better security against basic bots.

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge, etc.)
- No additional dependencies required

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Daniish-Qureshi/Captcha-Generator.git
   ```
2. Open `index.html` in your web browser

## Usage

1. A random CAPTCHA will be generated automatically when the page loads
2. Enter the displayed CAPTCHA code in the input field
3. Click the "Check" button to verify your input
4. Click "Generate New" to get a new CAPTCHA code

## Customization

You can easily customize the CAPTCHA generation by modifying the `script.js` file:

- Change the character set by modifying the `alphabets` string
- Adjust the CAPTCHA length by adding or removing characters in the `generate()` function
- Modify the styling in `style.css` to match your website's theme
